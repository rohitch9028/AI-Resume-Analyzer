# AI Resume Analyzer

AI Resume Analyzer is a web-based application built using Python and Streamlit that analyzes resumes and extracts important skills using NLP techniques.

---

## Features

- Resume PDF Upload
- Skill Extraction
- Resume Score Analysis
- ATS-style Keyword Matching
- Missing Skill Suggestions
- Most Common Word Analysis

---

## Technologies Used

- Python
- Streamlit
- NLP
- pdfplumber
- Regular Expressions
- Data Analysis

---

## How It Works

1. Upload Resume PDF
2. Extract Resume Text
3. Analyze Skills
4. Generate Resume Score
5. Display Suggestions and Insights

---
